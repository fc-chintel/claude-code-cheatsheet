Claude Code コマンドチートシート

ターミナルで Claude Code をはじめて使う人向け。よく使う操作をジャンル別・重要度順にまとめました。

用語だけ先に： 「セッション」＝Claude Codeとの1回の会話のこと。「スラッシュコマンド」＝会話中に / から入力する特別な命令のこと。
目次
  1. セッションの開始・再開（ターミナルで打つコマンド）
  2. 会話中によく使うスラッシュコマンド
  3. 覚えておくと捗るキー操作
  4. 作業を進める・見直すときに使うコマンド
  5. さらに使いこなしたい人向け（発展編）

1. セッションの開始・再開（ターミナルで打つコマンド）

Claude Codeを立ち上げる・過去の会話に戻るときに使う、最初に覚える系のコマンド。

コマンド短縮形これは何をする？
claude — 今いるフォルダで、新しい会話（セッション）を始める。
claude "やりたいこと" — 最初の指示を渡した状態で、新しい会話をいきなり開始する。
claude --continue claude -c 今いるフォルダの、一番最近の会話に一覧を挟まず即座に戻る。「さっきの続き」をしたいときに。
claude --resume claude -r 過去の会話の一覧（ピッカー）を表示し、そこから選んで開く。どの会話か選びたいときに。
claude --print "質問" claude -p "質問" 対話モードに入らず、1回だけ質問して答えを表示しすぐ終了する。ちょっと調べたいだけのときに便利。
claude --no-session-persistence — この回の会話を履歴に残さない（--printと組み合わせて使う）。見られたくない一回きりの作業のときに。

2. 会話中によく使うスラッシュコマンド

会話の中で / を入力すると出てくる操作。使用頻度が高いものから順に。

コマンドこれは何をする？
/clear 今の会話をリセットして真っさらな状態で新しい話題を始める。前の会話は消えず、後から --resume で開ける。
/compact 会話が長くなってきたときに、内容を要約して軽くする（同じ会話のまま続けたいとき）。
/help 使い方のヘルプを表示する。
/model 使うAIモデルを切り替える。何も指定せず実行すると選択メニューが出る。
/usage 今のセッションの利用料金や使用量を表示する（/cost でも同じ）。
/permissions ファイル変更やコマンド実行のたびに確認を求めるかどうかのルールを管理する。
/init このプロジェクト用の説明ファイル（CLAUDE.md）を作成する。新しいフォルダで最初にやると良い。

3. 覚えておくと捗るキー操作

マウスを使わず、キーボードだけでサッと操作できる。

キーこれは何をする？
Esc Claudeが応答・作業している最中に、途中で止めて指示を出し直す。
Ctrl + C 1回目は入力中のテキストを消す（実行中なら中断）。何もない状態でもう1回押すと終了する。
Ctrl + D Claude Codeを終了する。
Esc を2回 入力欄が空なら、以前の状態に戻す「巻き戻しメニュー」を開く（コードやり直しに便利）。
! を先頭に付ける そのままターミナルのコマンドとして実行する（例: !git status）。
@ ファイル名の候補が出てきて、会話中に特定のファイルを指定できる。
Shift + Enter 改行して複数行の指示を入力する（多くのターミナルでそのまま使用可）。
↑ キー 過去に入力したメッセージを呼び戻す。

4. 作業を進める・見直すときに使うコマンド

コードを書いてもらったあと、内容を確認したり、失敗をやり直したりするときに。

コマンドこれは何をする？
/rewind 会話やコードの変更を、以前の状態に巻き戻す。「やっぱり元に戻したい」ときの安全ネット。
/diff 今どんな変更が加えられたかを、差分表示で確認する。
/code-review 今の変更内容にバグや改善点がないか、Claudeにレビューさせる。
/doctor Claude Codeのインストールや設定に問題がないか診断する。うまく動かないときに。

5. さらに使いこなしたい人向け（発展編）

慣れてきたら覚えると便利。最初は読み飛ばしてOK。

コマンドこれは何をする？
/plan 大きな変更に着手する前に、計画だけ立てさせてから実行の許可を出す「プランモード」に入る。
/memory プロジェクトの説明ファイル（CLAUDE.md）を編集する。
/agents 特定の作業を任せる「サブエージェント」を作成・管理する。
/mcp 外部サービス（Slack、Googleカレンダーなど）と連携するMCPサーバーを管理する。
claude update Claude Code自体を最新バージョンに更新する。
claude --model <名前> 起動時に使うモデルを指定する（例: sonnet, opus）。
Claude Code チートシート — 個人利用メモ（公式ドキュメントを基に作成）
